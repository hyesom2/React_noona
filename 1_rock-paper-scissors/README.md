## 2. 기능
1. 게임방법버튼
- 게임방법버튼 클릭 시 게임방법이미지와 Overlay가 같이 나온다.
- X표시와 Overlay를 클릭하면 게임방법이미지는 사라진다.

2. 리셋버튼
- 리셋버튼 클릭 시 승패 카운트와 Player와 Computer의 Box안의 내용들도 같이 초기화 된다.

3. 가위바위보 버튼
- 해당 이미지버튼 클릭 시 Player의 Box에 출력된다.
- Computer의 Box는 랜덤으로 출력된다.
