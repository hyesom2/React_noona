## 2. 기능(?)
(1) 로그인 click -> 로그아웃으로 변경, OOO님 안녕하세요 출력 <br/>
(2) 로그아웃 click -> 로그인으로 변경 <br/>

(4) 장바구니에서 삭제 -> 장바구니옆 count가 1씩 빼짐, 0미만으로는 안내려가도록함 <br/>
